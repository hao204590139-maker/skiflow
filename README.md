# 🏔️ Skiflow 雪迹

**Skiflow** — 系统化滑雪学习平台。从零基础到顶级技巧，涵盖双板 ⛷️ 和单板 🏂 两条学习轨道，共 48 章精心编排课程 + 4 个实用附录，配合视频讲解。

## ✨ 功能

- 🎿 **双板** + 🏂 **单板** 两条完整学习路径
- 📚 5 个学习阶段，48 个系统章节，50+ 教学视频
- 🔍 全文搜索 (Ctrl+K)
- 🌙 深色/浅色主题
- 📱 手机/平板自适应
- 📊 学习进度追踪 (本地存储)
- ⌨️ 键盘快捷键翻章节

## 🚀 使用

**本机打开**：直接双击 `index.html`

**局域网/手机访问**：双击 `start-server.bat`，手机浏览器输入显示的 IP 地址

## 🌐 在线地址

部署后可通过以下地址访问：

- GitHub Pages: `https://<username>.github.io/skiflow`
- 自定义域名: 配置 CNAME 记录指向 GitHub Pages
